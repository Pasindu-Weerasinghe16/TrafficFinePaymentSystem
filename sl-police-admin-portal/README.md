# SL Police Admin Portal

React-based web portal scaffold for senior officials.

## Stable Versions

- Node.js: `24.16.0`

## Start Up

- Container start: `node server.js`
- Local start: `node server.js`

## Structure

- `src/components` - shared UI pieces
- `src/dashboard` - charts and analytics screens
- `src/services` - authenticated API gateway integration
