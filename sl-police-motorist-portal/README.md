# SL Police Motorist Portal

React-based web portal scaffold for motorists.

## Stable Versions

- Node.js: `24.16.0`

## Start Up

- Container start: `node server.js`
- Local start: `node server.js`

## Structure

- `src/components` - shared UI pieces
- `src/views` - fine search and payment screens
- `src/services` - API gateway integration
